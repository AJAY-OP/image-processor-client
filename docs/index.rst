Asynchronous Image Processor Client for python documentation.
=============================================================

.. image:: /images/background.jpg

An asynchronous python client for `Image Processor`__ API,
implementing all of the features and methods provided by it.

**Features**

- Non-blocking asynchronous wrapper
- Better object oriented approach.
- Implements entire methods provided by API.
- Clear categorisation of image methods into different objects.
